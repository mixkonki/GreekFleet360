# Product Context — GreekFleet 360

## Γιατί Υπάρχει Αυτό το Project

Οι ελληνικές εταιρείες μεταφορών δεν έχουν ολοκληρωμένα εργαλεία που συνδυάζουν:
- Διαχείριση στόλου (οχήματα, οδηγοί, συμμόρφωση)
- Οικονομική ευφυΐα (πραγματική κερδοφορία, όχι απλή παρακολούθηση εξόδων)
- Υποστήριξη αποφάσεων (actionable recommendations)

Τα υπάρχοντα εργαλεία είναι:
- **Πολύ γενικά**: Διεθνείς πλατφόρμες που δεν καταλαβαίνουν ελληνικές ρυθμίσεις
- **Πολύ κατακερματισμένα**: Ξεχωριστά εργαλεία για στόλο, οικονομικά, συμμόρφωση
- **Πολύ ακριβά**: Enterprise-only τιμολόγηση

## Τι Λύνει

### Λειτουργικά Προβλήματα
1. **Παρακολούθηση οχημάτων**: Ενιαίο model για όλους τους τύπους (φορτηγά → scooters)
2. **Καταγραφή καυσίμων**: Ακριβής υπολογισμός κατανάλωσης (μέθοδος full-tank)
3. **Συντήρηση**: Ιστορικό, κόστη, επόμενη συντήρηση
4. **Συμμόρφωση**: KTEO, ασφάλεια, ΠΕΙ/CPC, άδειες οδηγών

### Οικονομικά Προβλήματα
1. **Cost Engine v1**: Υπολογισμός πραγματικού κόστους ανά km/ώρα/δρομολόγιο
2. **Κερδοφορία εντολών**: Revenue - (vehicle cost + overhead) = profit + margin
3. **Ιστορικά snapshots**: CostRateSnapshot, OrderCostBreakdown
4. **KPI API**: Summary, cost structure, trend endpoints

## Πώς Λειτουργεί (User Flow)

### Manager (Διαχειριστής)
1. Dashboard → KPI cards (έσοδα μήνα, margin, ενεργά οχήματα, επερχόμενη συντήρηση)
2. Finance Settings → Διαχείριση εξόδων (CompanyExpense), κέντρα κόστους, υπάλληλοι
3. Orders → Δημιουργία/παρακολούθηση εντολών μεταφοράς
4. Fleet → Διαχείριση οχημάτων (CRUD)
5. API → Cost Engine analytics για BI dashboards

### Driver (Οδηγός) — Μελλοντικό
- Mobile PWA: Walk-around check, καταχώρηση καυσίμου, προβολή δρομολογίου

### Admin (SaaS Admin)
- Django Unfold admin panel
- Group 1: SaaS Platform (Companies, Users, Profiles)
- Group 2: Master Data (Expense Families, Categories, Θέσεις Εργασίας)
- Group 3: Tenant Data (View Only)

## UX Goals
- **Ελληνική γλώσσα** παντού στο UI
- **Ελληνικές μορφές**: DD/MM/YYYY, 1.234,56 €
- **HTMX**: Χωρίς page reloads, χωρίς SPA complexity
- **Tailwind CSS**: Utility-first, responsive
- **Leaflet.js**: Χάρτες για routes στις εντολές μεταφοράς

## Localization (Ελληνική Αγορά)
- KTEO (Τεχνικός Έλεγχος)
- Τέλη κυκλοφορίας (Teli)
- Τροχαία (κλήσεις)
- ΠΕΙ/CPC (Πιστοποιητικό Επαγγελματικής Ικανότητας)
- Βαθμοί άδειας (Σύστημα Σησάμι)
- Ταχογράφος (φορτηγά)
- ADR (επικίνδυνα εμπορεύματα)
- Συνταιρία (ταξί διπλής βάρδιας)
- 14 μισθοί/έτος (Ελληνικό εργατικό δίκαιο)
